## Vstupy a vystupy
- `SW` - switch (přepínače)
- `clk` - hodinový signál 
- `BTNC` - tlačítko čítače (counteru)
- `BTNR` - reset tlačítko 
- `AN` - anoda
- `SEG` - segmenty segmentovky
- `LED` - ledka
- `DP` - desetinná tečka
